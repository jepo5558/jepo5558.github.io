ShadowSync APK

아래 버튼은 항상 최신 debug APK를 다운로드합니다.

Latest version: 0.8.1 (versionCode 9)

최신 APK 다운로드
  • 스크립트 리스트를 스크롤해도 컨트롤러 영역이 유지됩니다.
  • Script 헤더에 All, None 버튼이 있습니다.
  • 기본값은 전체 선택입니다.
  • 마지막 스크립트만 선택한 경우의 반복 재생 이슈를 보정했습니다.

설치할 때 Android가 "알 수 없는 앱 설치" 권한을 요청할 수 있습니다. 이 APK는 개발용 debug 빌드입니다.
